# Carpeta `notion/`

Deja aquí tu export de Notion (el archivo `.zip` que descargas con
**Export → Markdown & CSV** desde tu base de datos de Notion).

El script toma automáticamente el `.zip` **más reciente** de esta carpeta.

> ⚠️ Los archivos `.zip` de esta carpeta están en `.gitignore`: tu contenido
> **no** se sube al repositorio. Esta carpeta se versiona vacía (solo este README).
